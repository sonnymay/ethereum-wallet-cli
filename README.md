# Ethereum Wallet CLI

Small CLI tool built with viem.

## What it does
- Connects to Ethereum mainnet
- Accepts a wallet address or ENS name
- Prints the wallet address
- Prints the ETH balance
- Prints the latest block number

## Example
```bash
node scripts/wallet-cli.mjs vitalik.eth
node scripts/wallet-cli.mjs 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045
```
