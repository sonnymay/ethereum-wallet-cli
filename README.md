# Ethereum Wallet CLI

Small viem-based CLI project for learning Ethereum reads.

## Features

### Week 1
- Connect to Ethereum mainnet
- Read latest block data
- Read ETH wallet balance
- Resolve ENS names

### Week 2
- Read ERC-20 token data
- Read ERC-20 wallet balances
- Read Uniswap V2 reserves
- Estimate token price in ETH
- Read recent Transfer events
- Handle invalid token contracts
